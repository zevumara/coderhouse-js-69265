MPA: Multi-Page Application

Las multi-page application (aplicaciónes de páginas múltiples) son el típico sitio web con diferentes archivos HTML por cada sección.

Por ejemplo, el proyecto final que hicieron para el curso de Desarrollo Web es una MPA.
